# Business

## Target User

Primary user:

- Small overseas Etsy seller.
- Solo operator or tiny shop.
- Sells physical or digital products.
- Uses Etsy CSV exports.
- Needs profit clarity for tax season, CPA review, or business decisions.

Secondary user:

- CPA or bookkeeper receiving the seller's report.

## Pain Point

Etsy sellers struggle to combine:

- Orders.
- Refunds.
- Etsy fees.
- Payment processing fees.
- Etsy Ads.
- Offsite Ads.
- Shipping labels.
- Sales Tax / VAT / GST.
- Deposits.
- Product cost and packaging cost.

The pain is not only calculation. The pain is confidence: sellers do not know which numbers should count toward profit and which taxes are pass-through items.

## Value Proposition

ProfitLens gives Etsy sellers a fast way to:

- Upload Etsy CSV files.
- See clean income, fee, tax, and profit breakdowns.
- Understand what is excluded from profit.
- Add COGS when available.
- Export a report for CPA review.
- Avoid spending hours manually organizing spreadsheets.

## Pricing

Default MVP pricing:

- Free preview.
- Free Excel / PDF download by default.
- No forced payment gate in the default MVP.

Future pricing experiments after usage validation:

- $19 one-time Etsy annual profit report.
- $49/year for single-shop access.
- $99/year for multi-report or multi-shop access.

MVP should start with the simplest free path:

> Free upload, free preview, free Excel / PDF download.

## Revenue Goal

First revenue target:

- First active users before adding payment friction.
- First dollar within 90 days through a later paid experiment, manual offer, or premium report option.
- Validate whether sellers find the free report valuable enough to return, recommend, or pay later.

## 90-Day Commercial Goal

Day 1-30:

- Ship MVP.
- Publish landing page.
- Test with sample Etsy CSV files.
- Invite first users manually.

Day 31-60:

- Collect seller feedback.
- Improve parser coverage.
- Add missing report lines only when users request them.
- Test pricing.

Day 61-90:

- Get first paid users.
- Improve onboarding and export quality.
- Decide whether to introduce a $19 one-time report, premium export, or subscription.
